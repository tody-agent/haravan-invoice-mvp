# Continuity Protocol

## Active Goal
Clone and refactor the Hilo E-Invoice product into a modern, high-quality UX web app.

## Current Phase
planning

## Next Actions
- [ ] 1.1 Scaffold modern frontend app (Next.js / Vite React) with TailwindCSS.
- [ ] 1.2 Implement core UI components (Sidebar, Topbar, Data Tables, Modals/Slide-overs) using Shadcn UI.
- [ ] 1.3 Create routing structure for all main modules.

## Working Context
- Key Decision: Use React + Tailwind + Shadcn UI.
- Key Decision: Consolidate Settings, Master Data, Invoices, and Reports into a cohesive sidebar-driven app.
- Key Decision: Use live previews and slide-overs to fix the clunky multi-page form experience of the original Hilo product.
- Status: PDF documentation from the original system is currently being downloaded and parsed by a multi-threaded Gemini CLI script into the `docs/` folder.

## Hard Rules
- Aesthetic quality must be top-tier.
- Do not skip planning phases.

## Recent Learnings
- Hilo's e-invoice system uses complex, fragmented pages. A unified dashboard approach will significantly improve UX.
