# Gói dịch vụ

> Xem thông tin gói dịch vụ hiện tại, usage limits (hóa đơn, storage) và danh sách tính năng available.

:::tip Tóm tắt
Settings Plan hiển thị current plan card, progress bars cho invoice usage và storage usage (đổi màu ở 80%), và feature grid với 6 tính năng.
:::

## Truy cập

- Sidebar: **Cấu hình** → **Gói dịch vụ**
- URL: `/settings/plan`

## Current plan card

Hiển thị:

- Icon gói dịch vụ
- Tên gói (Free/Pro/Enterprise)
- Label mô tả

## Usage progress bars

| Metric | Hiển thị |
|---|---|
| Invoice usage | Progress bar (đỏ khi ≥80%) |
| Storage usage | Progress bar (đỏ khi ≥80%) |

## Feature grid

6 tính năng với status available/locked:

| Tính năng | Status |
|---|---|
| Phát hành HĐ | ✅ Available |
| Báo cáo | ✅ Available |
| Xuất PDF | ✅ Available |
| Quản lý KH | ✅ Available |
| Tự động hóa | ✅ Available |
| AI Tiền-kiểm | 🔜 Coming soon |
